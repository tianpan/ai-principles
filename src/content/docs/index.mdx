## 如何使用本教程

1. **按顺序学习**：章节之间有依赖关系，建议从头开始
2. **动手实验**：每个交互组件都可以调整参数，尝试不同设置
3. **理解代码**：Lab 部分的代码可以直接运行，修改参数观察变化
4. **完成验收**：最终目标是让训练 loss 下降 30%+，生成 20+ 个 token
