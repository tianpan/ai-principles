# Mini Transformer Lab

这里是 Mini Transformer Lab，你将亲手从零实现一个最小可运行的 Transformer。

## 目标

1. 理解 Transformer 各模块的代码实现
2. 在 toy 数据集上完成训练
3. 使用训练好的模型进行文本生成

## 实验内容

- **代码实现**：从 Embedding 到输出层的完整代码
- **训练验证**：观察 loss 下降过程
- **推理演示**：给定 prompt 生成文本
