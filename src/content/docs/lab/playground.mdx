## 使用指南

### 🎮 游乐场模式

在这里你可以：

1. **训练模型** - 输入你自己的训练数据，观察模型如何学习
2. **生成文本** - 输入提示文本，让模型自动续写
3. **观察 Attention** - 实时查看生成过程中的注意力分布

### 🔍 Attention 模式

深入了解 Self-Attention 机制：

- 查看 Q、K、V 的含义和作用
- 交互式探索注意力矩阵
- 理解注意力权重如何影响输出

### 📈 训练模式

监控模型训练过程：

- 实时 Loss 曲线
- 训练进度统计
- 模型参数可视化

### 🏗️ 架构模式

探索模型结构：

- 各层的作用和连接
- 数据流动方向
- 可调整的参数

## 注意事项

- 首次加载需要下载 Pyodide（约 10MB），请耐心等待
- 训练数据越丰富，生成效果越好
- 建议使用 Chrome 或 Firefox 浏览器获得最佳体验
