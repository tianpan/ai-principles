# 德州扑克网页游戏

这是一个简单的德州扑克网页游戏，支持单人对战AI玩家。

## 功能特点

- 支持2-6名玩家（1名真实玩家和1-5名AI玩家）
- 完整的德州扑克游戏流程：发牌、翻牌、转牌、河牌、摊牌
- 支持基本操作：跟注、加注、让牌、弃牌
- 简单的游戏界面设计

## 如何运行

1. 将所有文件下载到您的计算机上
2. 在浏览器中打开 `index.html` 文件
3. 输入您的玩家名称，然后点击"开始游戏"

## 游戏规则

1. 每位玩家初始拥有1000筹码
2. 游戏自动为您随机分配1-5名AI对手
3. 每回合发给每名玩家2张底牌，并发5张公共牌（逐步展示）
4. 您可以选择弃牌、让牌、跟注或加注
5. AI玩家会自动做出决策
6. 游戏结束时，未弃牌的玩家中随机选择一名获胜者（简化版）

## 文件结构

- `index.html` - 游戏的HTML结构
- `css/style.css` - 游戏样式表
- `js/app.js` - 游戏逻辑和功能
