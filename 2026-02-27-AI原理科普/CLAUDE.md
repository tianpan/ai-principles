# AI 原理科普项目

## 目标

面向零基础学习者的生成式AI交互科普产品：从"看懂Transformer"到"从零实现最小Transformer"。

## 技术栈

Astro + Starlight + React + D3.js + Tailwind CSS

## 核心原则

1. **可视化优先**：每个概念必须有动态可视化（3Blue1Brown 风格）
2. **零基础可达**：概念 → 计算 → 实现三层递进，用类比代替术语
3. **可验证学习**：用户最终能从零实现最小 Transformer
4. **测试驱动开发**：所有新开发的内容都必须增加相对应的测试用例

## 技术规范

| 类型 | 技术选型 |
|------|----------|
| 组件 | React + TypeScript |
| 样式 | Tailwind CSS |
| 动画 | Framer Motion |
| 可视化 | D3.js |

## 视觉风格

- 配色：科技蓝 + 温暖橙
- 风格：简洁现代、3Blue1Brown 式动画美学

## 验收标准

1. 解释 Transformer 各模块"做什么、为什么需要"
2. 手动走通简化 attention 计算流程
3. 从零实现最小 Transformer 完成训练与生成
