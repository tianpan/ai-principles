# Component Upgrade Spec

## 功能描述

升级现有可视化组件，从静态升级为动态动画版本。

## 升级清单

### AttentionExplorer
**当前：** 静态热力图，点击显示数值

**升级后：**
- 热力图单元格入场动画（逐个淡入）
- 连线动画：Token → 热力图单元格
- Tooltip 显示详细解释
- 支持鼠标悬停高亮

### QKVStepByStep
**当前：** 分步展示，点击切换

**升级后：**
- 向量变换动画（平滑过渡）
- 矩阵乘法动画（数值流动）
- Softmax 动画（归一化过程）
- 自动播放模式

### MultiHeadComparator
**当前：** 静态对比，滑块切换

**升级后：**
- 头数切换时的过渡动画
- 各头矩阵入场动画
- 合并动画（多头 → 单头）

### PositionalEncoding
**当前：** 静态波形图

**升级后：**
- 波形动态生成动画
- 维度切换时的波形变化
- 3D 波形旋转（可选）

## 技术要求

- 使用 Framer Motion 实现动画
- 动画时长 300-500ms
- 支持 prefers-reduced-motion
- 保持组件 API 兼容

## 验收标准

- [ ] 所有组件动画流畅
- [ ] 移动端适配
- [ ] 无障碍支持
